🐛 TROUBLESHOOTING RÁPIDO:

❌ "Connection refused"
   → Asegúrate que la API está corriendo en Terminal 1

❌ "Missing environment variables"
   → Verifica que .env tiene todas las variables de Azure

❌ "ModuleNotFoundError: ragas"
   → pip install -r testeos/requirements_ragas.txt

❌ "Permission denied" en run_tests.sh
   → chmod +x /home/administrador/vodafone/testeos/run_tests.sh
